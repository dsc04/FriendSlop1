# ArtSource — исходники графики

Здесь лежат **исходные файлы** (`.blend`, `.psd` и т.п.), из которых
экспортируются ассеты для игры.

Папка находится **вне `Assets/`**, поэтому Unity её не импортирует —
`.blend` не сломает проект у тех, у кого не установлен Blender.

## Как работать
1. Модель делаем в Blender, `.blend` сохраняем сюда — в `ArtSource/Models/`.
2. Экспортируем **FBX** (`File ▸ Export ▸ FBX`) в `Assets/_Project/Art/Models/` —
   именно этот FBX использует игра.
3. Коммитим оба файла (подробнее — в `SETUP.md`, раздел «Для художников»).

> ⚠️ `.blend` — крупные бинарники, хранятся в Git LFS. На бесплатном тарифе
> GitHub лимит LFS — 1 ГБ на всех. Не коммитьте лишние тяжёлые версии.
